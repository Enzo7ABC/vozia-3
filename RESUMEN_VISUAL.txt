📊 RESUMEN DE LA ARQUITECTURA ACTUAL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✓ Orquestación de IA:       LangGraph (Flujos de estado cíclicos y dinámicos)
  ✓ Endpoints de Servicios:   FastAPI (Python 3.11+, puerto 8000)
  ✓ Interfaz del Operador:    React 18 + Vite + Tailwind CSS (puerto 3000)
  ✓ Motor NLP Local Fallback:  PyTorch + pysentimiento en CPU (Offline)
  ✓ Bases de Datos:           MongoDB Atlas (Nube) / calls_db.json (Local Fallback)
  ✓ Conectores de Modelos:    Ollama (Gemma 2B/Mistral local) & Groq Cloud (Llama cloud)
  ✓ Claves de Nube:           Rotación multi-llave para evitar cuotas de Rate Limit

🔌 ENDPOINTS DE LA API (FastAPI)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  • GET  /ia-voz/models         ➔ Lista los modelos de IA activos e instalados.
  • POST /ia-voz/call-state     ➔ Procesa y analiza la llamada por medio de LangGraph.
  • POST /copilot/chat          ➔ Ejecuta el enrutador inteligente y asiste al agente.
  • GET  /ia-voz/history        ➔ Recupera las llamadas guardadas.
  • GET  /ia-voz/audio/{id}     ➔ Obtiene el archivo de audio grabado en formato binario.
  • GET  /dashboard/data        ➔ Calcula métricas e indicadores de control para el dashboard.
